# Movie Review Sentiment Analysis using Deep Learning

This project implements an end-to-end deep learning pipeline for movie review sentiment
classification using the IMDb dataset. The workflow includes:

- Exploratory text data analysis
- Tokenization and sequence padding
- LSTM and Bidirectional LSTM model training
- Evaluation using accuracy, ROC-AUC, and confusion matrix
