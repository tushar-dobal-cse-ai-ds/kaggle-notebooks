# Forest Cover Type Classification

This project predicts forest cover type using machine learning models trained on cartographic variables.

## Dataset

Forest Cover Type Dataset (UCI / Kaggle)

Features include:

- Elevation
- Slope
- Hillshade
- Distance to Hydrology
- Distance to Roadways
- Distance to Fire Points
- Soil Type
- Wilderness Area

Target variable:
Cover_Type (7 forest classes)

## Models Used

- Random Forest
- ExtraTrees Classifier

## Final Model Accuracy

96.75%

## Key Techniques

- Feature Engineering
- Ensemble Tree Models
- Feature Importance Analysis
