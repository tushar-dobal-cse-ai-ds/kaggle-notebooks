# House Prices Prediction

## Overview
This project predicts house sale prices using a complete regression
pipeline. The workflow includes exploratory data analysis, feature
engineering, handling missing values, regularization, and model
evaluation using RMSE.

## Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- Random Forest Regressor

## Results
- Best model: Lasso Regression
- Public Kaggle RMSE score: **0.13138**

## Techniques Applied
- Log transformation of target variable
- Missing value handling using domain knowledge
- One-hot encoding of categorical variables
- Regularization and cross-validation
