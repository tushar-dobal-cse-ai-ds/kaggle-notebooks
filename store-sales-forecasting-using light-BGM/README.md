# 📊 Store Sales Forecasting using LightGBM

## 🚀 Overview

This project focuses on forecasting store sales using advanced machine learning techniques, specifically **LightGBM**, on time-series retail data.

The model captures temporal patterns, seasonality, and trends to generate accurate sales predictions.

---

## 🎯 Objective

To build a robust forecasting model that predicts future store sales using historical data and advanced feature engineering.

---

## 📊 Dataset

Dataset used:

* Store Sales Time Series Dataset
* Source: Kaggle

👉 Includes:

* Date-wise sales data
* Store and product information
* Time-based features

---

## 🧠 Approach

### 🔹 Feature Engineering

* Date features (year, month, day, day of week)
* Lag features
* Rolling statistics
* Trend and seasonality handling

---

### 🔹 Model Used

* **LightGBM Regressor**

Why LightGBM?

* Fast training
* Handles large datasets efficiently
* Excellent performance on tabular data

---

### 🔹 Evaluation

* RMSE (Root Mean Squared Error)
* Model comparison and validation

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* LightGBM
* Scikit-learn
* Matplotlib, Seaborn

---

## 📈 Results

* Achieved strong predictive performance using LightGBM
* Captured seasonal patterns effectively
* Improved accuracy through feature engineering
