# 🛍️ Customer Segmentation using Clustering

## 🚀 Overview

This project focuses on segmenting customers based on purchasing behavior using unsupervised machine learning techniques.

By applying clustering algorithms such as KMeans and DBSCAN, the model identifies meaningful customer groups that help businesses improve marketing strategies, customer targeting, and decision-making.

---

## 🎯 Objective

To segment customers into distinct groups based on spending behavior and income patterns, enabling data-driven business insights and personalized marketing strategies.

---

## 📊 Dataset

Dataset used:

* Mall Customer Segmentation Dataset
* Source: Kaggle

Dataset features:

* CustomerID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1–100)

The dataset is widely used for clustering and business analytics projects.

---

## 🧠 Machine Learning Approach

### 🔹 Exploratory Data Analysis

* Distribution of age, income, and spending score
* Customer behavior analysis
* Relationship between income and spending

---

### 🔹 Feature Engineering

Features selected for clustering:

* Annual Income (k$)
* Spending Score (1–100)

Feature scaling was applied using:

* StandardScaler

---

### 🔹 Models Used

### KMeans Clustering

* Elbow Method for optimal cluster selection
* Cluster visualization
* Silhouette Score evaluation

### DBSCAN

* Density-based clustering
* Noise detection for irregular customer patterns

---

## 📈 Evaluation

### Silhouette Score

Achieved strong cluster separation with meaningful customer grouping.

### Business Interpretation

Customer groups identified:

* Premium Customers
* Budget Customers
* Impulsive Buyers
* High Potential Customers
